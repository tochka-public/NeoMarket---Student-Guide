# 03 — Синдикаты

NeoMarket рухнул, но его подразделения — нет. Каждое из них стало **синдикатом**: автономной структурой со своей позицией, ресурсами и планом на будущее. Каждый синдикат уверен, что именно он знает, как собрать новый NeoMarket.

Ваша команда выбирает один синдикат на S0. Это ваша специализация на весь практикум: уникальные задачи, собственные контракты, навыки, которые пойдут в резюме.

---

## Базовые синдикаты

Три синдиката открыты с первого дня. Выбор происходит на S0.

### QA Corps

> *«Старый NeoMarket рухнул потому что никто не следил за качеством. Мы не повторим эту ошибку.»*

**Роль:** QA Engineer / SDET

QA Corps строит систему, которая ловит баги до того, как они дойдут до пользователей. Тест-планы, автотесты, нагрузочное тестирование — если что-то в NeoMarket может сломаться, QA Corps узнает об этом первым.

**Примеры задач:**
- Написать e2e-сценарии для ключевых user flows
- Поднять тестовую инфраструктуру для всех команд
- Провести нагрузочное тестирование API
- Создать тест-план для нового модуля

**Что в резюме:** автоматизация тестирования, e2e и нагрузочные тесты, тестовая инфраструктура. Спрос на рынке — стабильный, QA-инженеры нужны в каждой продуктовой команде.

---

### Interface

> *«Без интерфейса — нет пользователей. Без пользователей — нет маркетплейса.»*

**Роль:** Product Designer / UX Lead

Interface — синдикат с двойной ролью. С одной стороны, вы проектируете UI/UX: wireframes, user flows, дизайн-система, спецификации экранов. С другой — ваши спеки становятся заданиями для всех остальных команд. Вы одновременно и производитель, и заказчик.

Глава Interface — **отдельный преподаватель**, который ставит задачи, принимает работу и публикует готовые спеки как контракты для реализации.

**Примеры задач:**
- Спроектировать user flow для каталога товаров
- Создать UI-kit и дизайн-систему NeoMarket
- Подготовить wireframes для нового модуля
- Провести UX-аудит существующих экранов

**Что в резюме:** Product Design, UX/UI, дизайн-системы, спецификации интерфейсов. Спрос на рынке — высокий, продуктовые дизайнеры востребованы в IT и стартапах.

---

### Forge

> *«Инструменты, инфраструктура, фундамент. Пока остальные спорят — мы строим.»*

**Роль:** Platform Engineer

Forge создает инструменты, которыми пользуются все. Shared-библиотеки, генераторы кода, CLI-утилиты, мониторинг — всё, что делает разработку быстрее и надежнее. Пока другие синдикаты спорят о том, каким должен быть NeoMarket, Forge уже закладывает фундамент.

**Примеры задач:**
- Написать shared-библиотеку (API-клиент, модуль авторизации)
- Создать генератор boilerplate-кода для новых endpoint'ов
- Собрать CLI-утилиту для локальной разработки
- Поднять дашборд мониторинга микросервисов

**Что в резюме:** Platform Engineering, DevEx, shared-инфраструктура, тулинг. Спрос на рынке — взрывной рост, платформенные инженеры в топе спроса 2026 года.

---

## Синдикаты, которые откроются позже

Два синдиката пока недоступны. Они откроются для команд, которые наберут достаточно Rep. Следите за объявлениями.

### Firewall

> *«Декеры положили NeoMarket один раз. Кто-то должен убедиться, что это не повторится.»*

Безопасность: аудит, OWASP-чеклисты, поиск уязвимостей. Если декеры снова придут — Firewall будет готов.

**Что в резюме:** Application Security, аудит безопасности. Спрос на рынке — очень высокий, security-инженеры среди самых дефицитных специалистов.

### Neural Watch

> *«Будущее торговли — AI. Кто контролирует алгоритмы — контролирует рынок.»*

Продвинутый AI/ML: архитектура ботов, RAG-пайплайны, fine-tuning. Neural Watch работает на стыке данных и интеллекта — там, где обычные промпты уже не справляются.

**Что в резюме:** AI/ML Engineering, RAG, LLM-интеграции. Спрос на рынке — взрывной рост.

---

## Entrance Quest

Вступление в синдикат — не формальность. При присоединении команда выполняет **entrance quest** — обязательное вступительное задание синдиката. Без него доступ к гильдейским контрактам закрыт.

Entrance quest выполняется в начале практикума и знакомит команду с профилем синдиката: QA Corps напишет первый тест-план, Interface — первый wireframe, Forge — первый shared-инструмент.

---

## AI — для всех

Базовые AI-контракты доступны **каждой команде** независимо от синдиката: промпты для code review, AI-генерация тестов, AI-прототипирование UI, проверка кода на уязвимости через LLM. AI/LLM — это мета-навык, а не привилегия одного синдиката.

Neural Watch занимается только продвинутыми задачами: RAG, fine-tuning, архитектура ML-пайплайнов.

---

## Как выбрать

На S0 Captain заполняет мини-анкету: 1-й и 2-й выбор синдиката. Распределение по квотам происходит прямо на встрече.

Несколько вопросов, которые помогут определиться:
- **Нравится ломать и проверять?** QA Corps
- **Нравится проектировать и визуализировать?** Interface
- **Нравится строить инструменты для других?** Forge

Какой бы синдикат вы ни выбрали — это реальная инженерная роль, которая прокачивает навыки, востребованные на рынке.
