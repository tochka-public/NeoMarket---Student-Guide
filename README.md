## Я отстал, что делать?

Если пропустил начало или не понимаешь что происходит:

1. Найди свою команду и Captain-а → [Состав синдикатов](06-roster.md)
2. Напиши Captain-у в Telegram — узнай какой модуль взяла команда и что уже сделано
3. Прочитай [задание](07-service-development.md) и подключайся к работе
4. Если не в команде — пиши преподавателю

## Не пропустил ли я чего?

Чеклист того, что уже должно быть:
- [ ] Ты в команде и знаешь кто Captain
- [ ] Команда в синдикате (Forge / Interface / QA Corps) → [состав](06-roster.md)
- [ ] Команда выбрала модуль (B2B / Moderation / B2C)
- [ ] Есть GitHub-репозиторий команды с кодом сервиса
- [ ] Хотя бы часть эндпоинтов реализована и работает через `docker compose up`

Если что-то не отмечено — разбирайся с Captain-ом, не откладывай.

## Краткая справка

- **Rep** — репутация. Растёт за выполнение контрактов. Определяет итоговую оценку (T1 / T2 / T3)
- **Credits** — игровая валюта. Магазин, аукцион. Не влияет на оценку
- **Contract** — задание. Выполняешь → получаешь Rep и Credits
- **Captain** — тимлид команды. Координирует работу, связь с преподавателем
- **Syndicate** — специализация команды (Forge / Interface / QA Corps)
