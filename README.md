# NeoMarket — Student Guide

NeoMarket мертв. Платформа, через которую шла вся торговля кибер-города, уничтожена декерами. Протоколы стерты, данные скомпрометированы.

Синдикаты набирают людей. Ваша задача — собрать новый NeoMarket.

## Краткая справка

- **Rep (Reputation)** — ваша репутация. Растет за выполнение контрактов. Определяет итоговую оценку
- **Credits** — игровая валюта. Тратится в магазине. Не влияет на оценку
- **Contract** — задание. Берете, выполняете, сдаете
- **Syndicate** — ваша специализация (выбирается командой)
- **Captain** — тимлид команды
